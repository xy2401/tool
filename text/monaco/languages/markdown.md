# Monaco 文本与代码工具箱使用教程

欢迎使用基于 Monaco Editor 打造的全能文本与代码工具箱！本页面不仅是一个强大的代码编辑器，还集成了多种实用的开发者功能。

## 💡 核心特性

### 1. 智能语言识别
当你向编辑器中粘贴代码或拖拽文件时，工具箱会自动识别代码所属的语言（支持数十种常用语言），并自动进行语法高亮。你也可以在顶部的**Auto**下拉菜单中查看识别评分详情。

### 2. 文本差异对比 (Diff)
- 勾选功能区的 **文本对比** 即可进入差异对比模式。
- 支持 **并排 (Side-by-side)** 与 **行内 (Inline)** 两种展示视图。
- **独家功能**：在对比 JSON 时，你可以点击“JSON 属性自然排序”，它会自动按字母顺序对左右两侧的 JSON 键进行排序，完美解决因属性顺序不同而导致的无意义差异干扰！

### 3. 内容实时预览 & 运行
勾选功能区右侧的预览开关，即可在分屏中实时看到效果：
- **Markdown 预览**：实时渲染 Markdown 为 HTML。
- **HTML 预览**：直接渲染网页内容。
- **JS 运行**：将输入的 JavaScript 代码直接放入 Console 中执行，并在预览窗口中输出 `console.log` 的结果。
- **SVG / Base64 预览**：支持直接预览 SVG 矢量图与 Base64 编码的图片或多媒体。

### 4. 数据格式转换
当编辑器语言被识别或设定为特定格式时，会自动解锁对应的转换功能：
- **JSON ↔ YAML**：一键双向互转。
- **压缩为一行**：支持将格式化的 JSON 或 XML 代码压缩为极简的一行。
- **HTML 转 Markdown**：一键提取 HTML 正文并转换为干净的 Markdown 格式（支持撤销）。

### 5. 极致的状态记忆
你当前选择的主题、小地图开关、语言、对比模式等所有偏好设置，都会**实时保存并极其精简地反映在浏览器的地址栏中**。
你可以直接将当前网址**加入书签**或**分享给他人**，下次打开时，工具箱将以你最习惯的专属状态完美恢复！
